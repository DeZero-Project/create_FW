# [Variableインスタンスの計算方法(追加)] 改修設計仕様書

## 1. 改修の目的・背景
- **背景**: [Variableインスタンスと異なる型で演算可能にする際に除算と減算や累乗、負数への変換を行うことでより複雑な計算を可能にできると考えたため]
- **目的**: [特殊メソッドを追加しaddとmul同様の計算を除算・減算等で行えるようにする]

## 2. 変更内容一覧

| クラス名 | 関数/メソッド名 | 変更の概要 |
| :--- | :--- | :--- |
| `Variable` | `__sub__ `/ `__truediv__`/ `__neg__ `/ `__pow__` | 演算時における演算子のオーバーロードを行う特殊メソッドを追加 |
| `Variable` | `__rsub__ `/ `__rtruediv__`| 右項がVariableインスタンスだった場合に呼び出される特殊メソッドを定義|

## 3. 詳細設計（Before & After）

### 3.1 [Variable]
#### 変更前 (Before)
- [Variableインスタンスを減算/除算する際は必ずVariableインスタンス同士でしか演算が行えなかった]
- [伝播された値を負数に変える機能が不足していた]

#### 変更後 (After)
- **演算方法**: Variableとスカラ値を使用した除算/減算が行える状態となる

## 4. 影響範囲と整合性
- **関連1**: `Variable` 演算時のオーバーロードを行う `__sub__ `/ `__div__`/ `__neg__ `/ `__pow__` を定義
- **関連2**: `Variable` 関数が演算時点の右項だった場合も演算可能にするための `__rsub__`と`__rdiv__` を定義

## 5. テスト・検証項目
- [] Variableインスタンスとスカラの演算が行えるか確認
- [] Variableインスタンスを右項に設定した場合に演算が行えるか確認

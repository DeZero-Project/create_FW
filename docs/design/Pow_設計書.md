# [Pow] 設計仕様書

## 1. 概要

2つの入力の積を計算する演算クラス。

## 2. 処理フロー (__init__)

1. 引数を`c`を 受け取る。
2. `c`をインスタンス変数に格納する。

## 3. 処理フロー (Forward)

1. 引数に `x` を受け取る。
2. `x` と `self.c` を累乗する。
3. 結果を `y` として返す。

## 4. 処理フロー (Backward)

1. 引数に逆伝播の値 `gy` を受け取る。
2. 入力値をインスタンス変数から取り出し変数 `x` に格納。(`self.inputs[0].data`)
3. `self.c`を変数`c`に代入。
4. 変数`gx`を作成し`c * x ** (c - 1) * gy`の結果を代入。
5. `gx` を返す

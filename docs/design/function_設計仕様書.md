# [Function] 設計仕様書

## 1. 概要

すべての演算関数の基底となるクラス。入出力の `Variable` 間の接続を管理し、順伝播および逆伝播の共通インターフェースを提供する。

## 2. クラス定義

| 変数名 | 型  | 説明 | 備考 |
| --- | --- | --- | --- |
| `inputs` | `list` | 入力される `Variable` のリスト。 |  |
| `outputs` | `list` | 出力される `Variable` のリスト。 |  |

## 3. メソッド/関数 インターフェース

### `__call__(self, *inputs)`

* **役割**: 引数からデータを抽出し、`forward` を呼び出し、結果を `Variable` として返す自動化処理。

### `forward(self, xs)`

* **役割**: 継承先で実装。呼び出された場合は例外を返す。

### `backward(self, gys)`

* **役割**: 継承先で実装。呼び出された場合は例外を返す。

# [Div] 設計仕様書

## 1. 概要

2つの入力の商を計算する除算クラス。

## 2. 処理フロー (Forward)

1. 引数に`x0` と `x1` を受け取る。
2. `x0` と `x1` を除算する。
3. 結果を `y` として返す。

## 3. 処理フロー (Backward)

1. 引数に逆伝播の値 `gy` を受け取る。
2. 入力値をインスタンス変数から取り出し変数 `x0` と `x1` に格納。(`self.inputs[0].data, self.inputs[1].data`)
3. 変数`gx0` を作成し`gy`と`x0` を除算した値を格納。
4. 変数`gx1` を作成し`gy`と`(-x0 / x1 ** 2)`を乗算した積を格納する。
5. `gx0` と `gx1` を返す。

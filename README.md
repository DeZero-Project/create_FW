# [プロジェクト名：例 Deep Learning FW from Scratch]

Pythonでスクラッチから構築しているディープラーニングフレームワークです。
理論の理解だけでなく、低レイヤのメモリ管理まで踏み込んで実装しています。

## 💡 このプロジェクトの核心（こだわり）
- **設計書先行型開発**: すべての機能を .md 形式の設計書に起こしてから実装（設計力重視）。
- **手動メモリ管理**: NLPモデル実装時に発生したGPUメモリリークを、Pythonレベルでの手動メモリ解放により解決。
- **C言語的視点**: ポインタの概念を意識し、計算グラフの参照管理を最適化。

## 🛠 技術スタック / 習得中のスキル
- Python (FW構築, NLP, 画像学習)
- C (ポインタ・メモリ管理)
- 設計 (要件定義, 設計書作成)

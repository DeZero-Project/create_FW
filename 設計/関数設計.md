# Functions/Classes
 ## Variable
  ### init
  - 引数にndarrayを指定
  - 引数からデータの抽出を行いメンバ変数に格納する
  - 勾配(grad)、変数の作成関数(creator)、データの中身(data)を取り出す
  - ndarray以外の値が送られた場合例外としてTypeエラーを返す
  ### set_creator
  - 変数を作成した関数を記憶する
  ### backward
  - 他の関数(creator)をリストに取り出しbackward処理の自動化を行う
  #### Variables
  - gys: 関数に格納された勾配をリスト形式で格納
  - gxs: gysに格納された勾配を関数のバックワードに渡した結果を格納
  - gx: for文で使用する。逆伝播時に出力層から送られる微分を格納

 ## Function
  ### call
  - 引数で渡された値(data)を抽出
  - Variable関数への受け渡しを行う
  - 継承先関数のforwardへの受け渡の自動化を行う
  ### forward/backward
  - Functionのforward/backwardが呼び出された場合例外を返す

 ## Square
  ### forward
  - 引数を2乗して返す
  ### backward
  - 引数で勾配を受け取る
  - メンバ変数から入力値(x)を取得
  - 引数の勾配(grad)と入力値(x)と2を掛けて返す
 
 ## Exp
  ### forward
  - 引数をnp.expに渡した結果を返す
  ### backward
  - 引数に勾配(grad)を受け取る
  - メンバ変数から入力値(x)を取得
  - 入力値(x)をnp.expに渡した結果と勾配(grad)を掛けて返す
  
 ## Helper Functions
  ### as_array
  - 引数にスカラー値を受けとりndarrayに変換する
     * numpyの仕様におけるTypeを回避する際に使用
  
  ### numerical_diff
  - 引数にインスタンス化した関数(f)、入力値(x)を受け取り定数のeps=1e-4を設定
  - 入力値(x)のデータを取り出しepsを足してVariable関数に渡す
  - 入力値(x)のデータを取り出しepsを引いてVariable関数に渡す
  - Variable関数に渡した結果を関数(f)に渡す
  - 関数(f)の結果の値(data)を使い数値微分を計算して返す

  ### square
  - Square関数のインスタンス化を行う

  ### exp
  - Exp関数のインスタンス化を行う
  ### add
  - Add関数のインスタンス化を行う
